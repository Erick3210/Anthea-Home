¿PARA QUÉ FINES UTILIZAREMOS SUS DATOS PERSONALES?

Los datos personales que recabamos de Usted, los utilizaremos para actividades orientadas con el objeto de RTD, como son:

  1. Conformar una base de datos con fines estadísticos, que nos permitan determinar cantidad de personas afines o comprometidas con los fines de RTD;
  2. Convocarle a eventos y/o actividades que realiza RTD con motivo del impulso a la concientización colectiva que promueve;
  3. Recabar datos personales para el llenado de formularios, así como de contacto que únicamente serán utilizados para fines estadísticos, así como para responder a sus inquietudes en lo particular;
  4. La información proporcionada sólo podrá ser utilizada para fines estadísticos que se elaboren para el seguimiento de avances y/o reportes por parte de RTD.

El uso de cualquiera de las herramientas, aplicaciones o medios de contacto contenidos en nuestra página web será considerado como el otorgamiento tácito del consentimiento para el uso de la información que nos comparta.
